# ⚡ Neon Type Racer

A fast, gamified typing game where **your car accelerates as you type**. Type the quote correctly to floor the throttle, race three AI rivals to the finish line, and level up to face faster competition. Built as a single self-contained HTML file — no installs, no dependencies, works offline.

> 🏁 **Play it live:** _(add your GitHub Pages link here once it's published)_

## ✨ Features

- **Speed = how fast you type.** Your 🏎️ surges forward with a glowing trail as your live WPM climbs.
- **AI rivals that scale with you.** Beat a race to level up; every level makes the bots faster, so the challenge always matches your skill.
- **Accuracy matters.** TypeRacer-style mechanics — a wrong key stalls your car until you fix it, so clean driving wins races.
- **Real quotes** to practice meaningful typing with natural punctuation and rhythm.
- **Three race lengths:** Sprint (1 quote), Rally (2), Marathon (3).
- **Progression & stats:** persistent levels and tiers (Rookie → Racer → Pro → Champion → Legend), win/race counts, daily streaks, top-speed and best-combo records — all saved locally in your browser.
- **Juice:** combo multiplier with a flame meter, podium leaderboard, weak-key drill tips, and optional engine sound.

## 🎮 How to play

1. Open the game (the live link above, or just open `index.html` in any browser).
2. Start typing — the timer and your engine start on the first keystroke.
3. Type each character correctly to move; mistakes stall you.
4. Cross the finish line **1st** to level up. Press **Esc** anytime for a new race.

## 🛠️ Tech

Plain HTML, CSS, and vanilla JavaScript in one file. No build step, no frameworks. Just open it.
